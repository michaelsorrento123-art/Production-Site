3. Description of the System

Instagram is a social media platform that allows users to share photos, videos, and messages. Creating an account requires users to enter identifying information such as an email address or phone number and a password. This sign-up process allows Instagram to create user profiles and provide access to its social networking features. The collected data supports account management, personalization, and security.

4. Data Collected

Visible form fields

  • Email address or phone number
  • Full name
  • Username
  • Password

Inferred data likely collected

  • IP address
  • Device type and operating system
  • Browser information
  • Approximate location

Behavioral data tracked after sign-up

  • Likes, comments, and follows
  • Time spent viewing posts
  • Search activity
  • Interaction with advertisements

5. Conceptual Database Structure

Primary Table: users

  • user_id (primary key)
  • email
  • username
  • hashed_password
  • full_name
  • created_at
  • last_login

Related Table: activity_log

  • activity_id (primary key)
  • user_id (foreign key)
  • activity_type
  • content_id
  • timestamp

Relationship explanation: The user_id field links the users table to the activity_log table, allowing the system to associate each recorded activity with a specific user.

6. Ethical Analysis

One ethical concern is privacy, as Instagram collects personal and behavioral data that can reveal detailed information about a user’s habits and preferences. This can cause me to think in ways it wants me to think based off of the information it wants to feed me. Another concern is surveillance, since users may not fully understand the extent of data tracking that occurs after sign-up. If this data were breached or misused, users could face identity theft, manipulation, or loss of privacy. These risks increase when data is retained long-term or shared with third parties.

7. Recommendations

  • Collect fewer behavioral data points by default.
  • Use clearer and more transparent consent language during sign-up.
  • Limit third-party access to user data.
