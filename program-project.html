Printing Instructions

Format: 4 Pages Total (2 Sheets), Double-Sided, Folded Lengthwise

Before You Print

  • Paper Size: 8.5” x 11” (Letter)
  • Total Pages in File: 4 pages
  • Print as: Actual Size (100%)
  • (Do NOT select “Fit to Page” or scaling options.)

Printer Settings

  • Open the file and select Print
  • Choose:
    • Print on Both Sides
    • Flip on Short Edge
    • (This is VERY important — it allows proper lengthwise folding.)
  • Pages: Print All Pages
  • Copies: 1 (or as needed)

After Printing

You will have:

  • 2 sheets of paper
  • Each sheet printed on both sides

Now:

  • Stack the pages in numerical order (1–4)
  • Fold each sheet lengthwise (hot-dog style)
  • Nest the folded sheets together
  • Crease firmly along the fold
